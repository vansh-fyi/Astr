# Component Inventory

## Mobile App Components

### Core Widgets (`lib/core/widgets/`)
Reusable, low-level widgets used across the application.

| Component | Description | Usage |
|-----------|-------------|-------|
| **AstrRiveAnimation** | Wrapper for Rive animations | Used for dynamic icons and illustrations |
| **GlassPanel** | Frosted glass effect container | Primary card background for UI elements |
| **GlassToast** | Custom toast notification with glass effect | In-app notifications and alerts |
| **RedModeOverlay** | Red tint overlay for night vision preservation | Toggled via settings for stargazing mode |

### Common Widgets (`lib/common/`)
Shared UI patterns specific to the Astr design system.

| Component | Description | Usage |
|-----------|-------------|-------|
| **AppBarGone** | Invisible app bar utility | Used to manage status bar color without taking space |
| **GridItem** | Standard grid layout item | Used in dashboard and catalog grids |
| **LinkCard** | Card with navigation behavior | Used for external links or internal navigation |
